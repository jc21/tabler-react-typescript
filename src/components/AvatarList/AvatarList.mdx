# AvatarList

Create a list of avatars within one parent container.

## Usage

<Playground>
	<AvatarList>
		<Avatar initials="JC" /> <Avatar initials="BV" />
		<Avatar initials="DH" /> <Avatar initials="RG" />
		<Avatar initials="LR" /> <Avatar initials="MD" />
		<Avatar initials="LR" color="pink-lt" />
		<Avatar initials="MD" color="red-lt" />
		<Avatar initials="JC" color="orange-lt" />
	</AvatarList>
</Playground>

## Stacked

Make the list stack once a certain number of avatars is reached to make it look
clear and display well regardless of the screen size.

<Playground>
	<AvatarList stacked>
		<Avatar initials="JC" /> <Avatar initials="BV" />
		<Avatar initials="DH" /> <Avatar initials="RG" />
		<Avatar initials="LR" /> <Avatar initials="MD" />
		<Avatar initials="LR" color="pink-lt" />
		<Avatar initials="MD" color="red-lt" />
		<Avatar initials="JC" color="orange-lt" />
	</AvatarList>
</Playground>
