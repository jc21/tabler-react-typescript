# Alert.Link

Add a link to your alert message to redirect users to the details they need to
complete or additional information they should read.

## Usage

<Playground>
	<Alert type="success">
		This is a success alert — <Alert.Link>check it out</Alert.Link>!
	</Alert>
	<Alert type="info">
		This is a info alert — <Alert.Link>check it out</Alert.Link>!
	</Alert>
	<Alert type="warning">
		This is a warning alert — <Alert.Link>check it out</Alert.Link>!
	</Alert>
	<Alert type="danger">
		This is a danger alert — <Alert.Link>check it out</Alert.Link>!
	</Alert>
</Playground>
