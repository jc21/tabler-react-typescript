# Flag

Thanks to the Tabler flags plug-in, you can create flags to visually represent
countries or languages. Flags are often used in forms, as an element of a
delivery address, phone number dialling code and many more.

This part of Tabler is distributed as plugin. To enable it you will need to
include additional css to your page:

```html
<link
	rel="stylesheet"
	href="https://unpkg.com/@tabler/core@1.0.0-beta2/dist/css/tabler-flags.min.css"
/>
```

## Usage

To create a flag choose the country whose flag you want to use. The full list of
countries can be found below.

<Playground>
	<Flag country="au" className="me-1" />
	<Flag country="nz" className="me-1" />
	<Flag country="us" className="me-1" />
	<Flag country="tg" />
</Playground>

## Sizes

Using Bootstrap’s typical naming structure, you can create a standard flag, or
scale it up or down to different sizes based on what’s needed.

<Playground>
	<Flag country="au" size="xl" className="me-1" />
	<Flag country="au" size="lg" className="me-1" />
	<Flag country="au" size="md" className="me-1" />
	<Flag country="au" size="sm" className="me-1" />
	<Flag country="au" size="xs" />
</Playground>
