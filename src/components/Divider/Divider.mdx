# Divider

Dividers help organize content and make the interface layout clear and
uncluttered. Greater clarity adds up to better user experience and enhanced
interaction with a website or app.

## Usage

Use dividers to visually separate content into parts. You can use a line only or
add text that will be centered by default.

<Playground>
	<Divider text="See also" />
</Playground>

## Alignment

You can modify the position of the text which is to be included in a separator
and make it left- or right-aligned. Otherwise, the text will remain centered.

<Playground>
	<p className="text-muted">
		Lorem ipsum dolor sit amet, consectetur adipisicing elit. Alias, dolore
		dolores doloribus est ex.
	</p>
	<Divider text="Chapter 1" align="left" />
	<p className="text-muted">
		Dicta error hic illo iure necessitatibus nihil officiis omnis
		perferendis, praesentium repellendus rerum, saepe sed, sit!
	</p>
	<Divider text="Chapter 2" />
	<p className="text-muted">
		Lorem ipsum dolor sit amet, consectetur adipisicing elit. Alias, dolore
		dolores doloribus est ex.
	</p>
	<Divider text="Chapter 3" align="right" />
	<p className="text-muted">
		Dicta error hic illo iure necessitatibus nihil officiis omnis

</Playground>
